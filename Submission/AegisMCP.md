# Elite Coders Open Source Hackathon Submission

## Project Details

- **Participant Name:** Akul Bansal
- **Team Name:** Power Solo
- **Project Title:** AegisMCP
- **Project Description:** Cloudflare for AI Agents. A zero-trust security proxy that sits between an AI Agent (like Claude) and the Model Context Protocol (MCP) servers it connects to. It features a tamper-proof cryptographic audit ledger, an AES-256 credential vault, and a cinematic Security Operations Center (SOC) dashboard.
- **Tech Stack:** Node.js, TypeScript, React, Vite, Tailwind CSS, SQLite, Express, MCP SDK v1.29
- **GitHub Repository Link:** https://github.com/Akulbanxal/AegisMCP
- **Demo / Deployment Link:** https://Akulbanxal.github.io/AegisMCP/
